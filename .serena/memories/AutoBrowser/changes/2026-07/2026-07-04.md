# Changes 2026-07-04

## Auto-update system
- New `src/AutoUpdater/` — AOT console EXE for file swap, SHA256 validation, rollback, relaunch
- New `src/AutoBrowser/Services/UpdateService.cs` — GitHub release check + download + install
- Post-build `CopyUpdater` MSBuild target copies `AutoUpdater*` into main app output
- `MainViewModel.CheckForUpdateAsync` — manual button with full status feedback
- `CheckForUpdateSilentAsync` — fire-and-forget on startup, only shows UI when update available
- `ShowUpdateDialogAsync` — reusable Yes/No dialog via `Wpf.Ui.Controls.MessageBox` (500px wide)
- Removed third Cancel button from update dialog

## CI pipeline
- Revision number: `[Math]::Floor(UTC_seconds * 65535 / 86400)` — full UInt16 range
- Tag builds: `MAJOR.MINOR.PATCH.REV`, branch builds: `1.MINOR.PATCH.REV`
- Updater publish step: `dotnet publish src\AutoUpdater\AutoUpdater.csproj`

## Build dependency
- AutoBrowser.csproj references AutoUpdater with `ReferenceOutputAssembly="false"`
- `CopyUpdater` target copies `AutoUpdater*` from `..\AutoUpdater\bin\$(Configuration)\net10.0\`

## Renames
- Folder: `src/AutoBrowser.AutoUpdater` → `src/AutoUpdater`
- Assembly: `AutoBrowser.Updater` → `AutoUpdater` (all references updated)

## Bug fixes
