# Changes 2026-07-14

## Notification Setting Additions
- Added `ShowPushNotifications` option (defaulting to true) to `AppSettings` model, `SettingsViewModel`, and `SettingsPage.xaml` view.
- Allowed users to enable/disable tray/balloon push notifications dynamically from the Settings UI.
- Integrated the setting verification in `SettingsViewModelTests`.

## Notification Setting Integration in URL Routing
- Updated routing methods in `App.Routing.cs` and `HomeViewModel.cs` to check the `ShowPushNotifications` configuration value before calling `ShowNotification`.

## Balloon Notification Fixes
- Replaced the instantaneous disposing/using pattern of `NotifyIcon` with a deferred disposal logic. The icon is kept alive for 4 seconds to allow the OS to display the balloon tip, preventing it from disappearing immediately upon object disposal.

## Pre-Push Hook Updates
- Updated `.husky/task-runner.json` pre-push task to run all tests (including UI tests) instead of filtering out UI tests.
